CS 456: Programming Languages


The undergraduate programming language class covers both practical and theoretical aspects of the discipline. Students will learn some of the key principles of programming languages and will gain a better understanding of the strengths and limitations of the languages that they are using. The class will make it easier to learn new languages and assess the suitability of a language for a given task.


TOPICS: Syntax and Semantics of Programming Languages, Implementation techniques, Abstract data types, Type checking,  Garbage collection, Object-Oriented languages, Functional languages, Scripting languages, Concurrency, Exception handling.
